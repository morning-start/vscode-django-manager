# django-manager

this is a django manager command just `ctrl+shift+p` and input `django-manager`

## Warning

**!!!** this extension need your default terminal to be `powershell`.

## Features

![alt text](images/feature.png)

### basic

startproject
startapp
makemigrations
migrate
runserver
